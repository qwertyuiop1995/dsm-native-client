# 产品路线图

## M0：工程基础与登录

- 单仓库、文档和协议契约。
- 三端工程骨架。
- HTTPS、证书信任和 NAS 配置。
- API 能力发现。
- 密码、OTP、SID 和 SynoToken。

## M1：只读文件客户端

- 共享文件夹和目录分页浏览。
- 文件详情和缩略图。
- 图片、文本和 PDF 预览。
- 单文件下载和传输中心。

## M2：文件写入

- 单文件上传和重名处理。
- 写权限检查。
- 安全删除、进度和结果校验。
- 删除后的回收站可恢复性检查。

## M3：回收站恢复

- 浏览 `#recycle`。
- 原始路径候选计算。
- `SYNO.FileStation.CopyMove` 恢复验证。
- 冲突保护和恢复结果校验。
- DSM File Station 专用恢复动作抓包研究。

## M4：五平台发布

- macOS、iPhone、iPad、Android 和 Windows P0 功能对齐。
- 平台构建、签名和 GitHub Actions。
- 安全、性能、兼容和无障碍检查。
- 公开测试版本。

## 后续候选

- 搜索、收藏、创建目录、重命名、复制和移动。
- 多文件 ZIP 下载和目录上传。
- 音视频预览和断点续传。
- 分享链接。
- Download Station、Photos、Docker 和 VMM。
