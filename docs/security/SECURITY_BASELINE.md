# 安全与隐私基线

## 必须遵守

- 只通过 HTTPS 连接 DSM。
- 不持久化明文密码和 OTP。
- SID、SynoToken 和 DID 使用平台安全存储。
- 不在 URL、日志、崩溃报告和分析平台中记录会话秘密。
- 自签名证书使用主机绑定和 SHA-256 指纹确认。
- 已信任证书变化时阻止静默连接。
- 文件名、路径、缩略图、预览和内容都视为隐私。

## 禁止提交

```text
真实 NAS 地址、IP 和 QuickConnect ID
用户账号、密码和 OTP
SID、SynoToken、Cookie 和 DID
证书私钥、签名密钥和配置文件
未脱敏 HAR、PCAP 和 DSM 响应
真实共享名、路径、文件名和用户文件
```

## 删除与恢复

- 删除前显示设备和目标摘要。
- 是否可恢复必须根据回收站实际状态验证。
- 回收站内删除视为永久删除并加强确认。
- 恢复冲突默认不覆盖。
- 超时后查询状态，不自动重复写操作。

## 发布检查

- Release 不包含抓包 CA、代理配置和 Debug 网络日志。
- 安装包不包含测试账号、临时响应和本地配置。
- 安全存储和预览缓存不进入系统云备份。
