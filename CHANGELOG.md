# 变更记录

本项目使用语义化版本，并在里程碑版本后附加阶段标识。

## Unreleased

### 新增

- 初始化单仓库目录结构。
- 添加 DSM API 参考和第一阶段开发文档。
- 添加进度、架构、安全和兼容性文档体系。
- 添加协议契约目录和初始 JSON Schema。
- 添加 GitHub 仓库结构检查工作流。
- 确定应用名称、跨平台包标识和 macOS 参考平台。
- 初始化 macOS SwiftUI App、Xcode workspace 与 Apple 共享 Swift Package。
- 实现 `DsmCore`、`DsmNetwork`、API 能力发现、密码/OTP 登录和 Keychain 会话存储。
- 添加 Apple 单元测试与构建工作流。
- 添加 UI UX Pro Max 自动路由、新设备引导安装脚本和离线测试。
- 完成 macOS 三栏原生文件客户端和传输中心。
- 实现多 NAS 配置、会话恢复、退出、自签名证书指纹信任和证书变化阻断。
- 实现 File Station 共享/目录/详情/缩略图、上传、下载、删除和移动 API Adapter。
- 实现图片、文本和 PDF 安全预览，以及回收站浏览和受兼容开关保护的恢复流程。
- 增加回收站路径、文件解析和二进制下载测试。
- 传输中心显示已传输大小、总大小、实时速度、完成百分比和预计剩余时间。
- macOS 支持基于 HTTP Range 的音乐和视频流式预览，无需先下载完整媒体文件。
- macOS 文件管理支持 Finder 风格快捷键、公开 API 重命名、拖拽移动及 10 秒限时撤销。
- 图片预览支持前后切换、左右旋转、滚轮缩放和常用预览快捷键。
- 文件管理支持在 NAS 上创建 ZIP/7z 压缩包，以及解压 ZIP、7z、RAR、TAR、GZ、BZ2 和 ISO 等常见格式；任务可在传输中心查看和取消。
- 大文件预览在网络读取期间显示实时速度；侧栏底部显示局域网、外网或 QuickConnect 连接，并提供带任务保护的退出入口。
- 最近访问支持单击选中、双击打开、逐条移除和一键清空；设置页新增本地数据与缓存占用查看和安全清理。
- 文件夹读取和搜索期间显示阻止重复操作的半透明加载遮罩；无法预览状态改为紧凑卡片。
- `.TS/.ts` 文件通过文件内容区分 MPEG 传输流与 TypeScript，不再根据文件大小猜测；系统播放器不支持编码时提示下载后使用其他播放器打开。
- 图片和视频进入全屏后隐藏标题、文件信息与底部操作栏，使用黑色背景完整展示内容。
- 图标和列表视图均支持拖拽移动；可写文件夹在成为落点时显示高亮与描边，图标视图支持在空白处拖动框选多个项目。
- 常见文本和代码文件支持应用内编辑并保存回 NAS；JSON、GeoJSON、XML、JavaScript、TypeScript 和 CSS 支持安全整理格式，关闭含未保存修改的窗口前会要求确认。
- 文件浏览器默认使用图标文件视图；不支持的视频只提示下载后使用其他播放器，不再为了外部播放自动下载到临时目录。

### 变更

- 客户端中文品牌名称统一为“岚仓”，英文品牌名称统一为 `LanStash`。
- macOS 显示名称、窗口标题和安装产物名称改用新品牌，兼容性技术标识保持不变。
- macOS 界面文案改为面向普通用户，并为错误提示补充可执行的恢复建议。
- 上传按钮改为直接打开文件选择器，搜索框提示精简为“搜索文件”。
- 设置页移除回收站只读说明和重复的退出登录按钮。

### 修复

- 复制或移动项目时，“粘贴”不再提前要求选择跳过或替换；仅在目标文件夹确实存在同名项目时询问，并在用户选择前不执行任何文件操作。
- 修复文本覆盖保存后再次打开仍显示空文件的问题；保存后同步最新文件大小，预览前也会重新读取文件详情，并移除底部重复的编辑工具栏。
- 修复深层目录因权限展示字段误报而无法拖拽移动的问题；移动前改用 NAS 正式权限检查。
- 修复列表视图中单个文件和文件夹无法开始拖拽的问题。
- 文本编辑器获得焦点时不再响应空格关闭预览等文件浏览快捷键，复制、粘贴、撤销和保存仍由系统编辑器处理。
- 修复空白文本文件无法显示编辑入口的问题；常见文本文件在预览底部提供明确的“编辑文件”按钮。
- 重命名不再被不可靠的文件写入标记提前禁用，改由 NAS 在执行时校验父目录权限。
- 从传输中心或设置返回时恢复之前浏览的目录，侧边栏“文件浏览器”也不再重置当前路径。
- 修复 DSM 接受账号密码后不识别 File Station SID 的兼容问题；会话现在同时通过安全 Cookie 请求头和 POST 参数传递。
- macOS 打包成功并验证新 DMG 后自动清理旧版本安装包。
- 修复登录后首次读取文件失败时自动执行退出、覆盖真实错误提示的问题；权限不足和会话失效现在会分别处理。
- 修复把 QuickConnect ID 当作普通主机名导致无法连接的问题，并支持直接粘贴浏览器最终地址。
- 修复家庭 NAS 使用 IP、短主机名或 `.local` 地址时，因地址与自签名证书名称不一致而无法进入首次信任流程的问题。
- 修复部分 HTTPS 证书挑战未稳定进入用户核对流程的问题；过期证书和证书变化仍会被阻止。
- 修复切换页面后下载失去本地保存位置访问权限、在接近完成时失败的问题。
- 上传请求按 File Station 官方要求发送完整数据长度，并为数据不完整、超时、同名冲突等错误提供明确提示。
- 修复视频播放器把 MIME 当作统一类型标识、未限制分段响应大小而无法播放大文件的问题。
- 音乐预览改为按需缓冲，优化 QuickConnect 等较慢网络下的首次播放和拖动体验。
- 解压前读取压缩包目录并自动选择中文文件名编码，修复旧版 ZIP 解压后文件名乱码；仅在压缩包确实加密时询问密码，密码错误可持续重试。
- 压缩任务的权限错误会区分“当前文件夹不可写”和“NAS 拒绝压缩”，并提示个人文件夹与共享文件夹权限不会因管理员身份自动绕过。
