# Android 原生客户端

计划使用 Kotlin、Jetpack Compose、Coroutines、OkHttp、Keystore、Room 和 WorkManager。

预定模块：

```text
app
core-domain
core-network
core-storage
feature-auth
feature-files
feature-transfer
feature-recycle
```

已确定中文应用名为“岚仓”、英文应用名为 `LanStash`，applicationId 保持为 `io.github.qwertyuiop1995.dsmnativeclient`，最低 Android 版本为 API 29。Gradle 工程将在 Android 实现批次初始化。
